Simulates collision of boxes, that calculates pi accurately if the mass of the bigger box is 100^n times bigger that the small one. 
The number of collisions divided by 10^n is equal to pi, accurate by n+1 digits. Due to framerate limitations, calculations further that n=3 are likely impossible/inadequate.
To compile you need gcc and raylib, the compiled file (game.exe) is a stable version with n=3.
